Twee dagen na de algemene beschikbaarheid van Visual Studio 2019 voor Windows en Mac stelt Microsoft Visual Studio Code beschikbaar op Linux als Snap. Deze versie bestaat al sinds 2016 voor Windows, Linux en Mac en geniet nu van Snapondersteuning voor naadloze automatische updates. Microsoft heeft Visual Studio 2019 eerder deze week vrijgegeven voor zowel [&#8230;]
