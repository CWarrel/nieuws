Mendix heeft twee nieuwe ontwikkelomgevingen gepresenteerd: Studio en Studio Pro. Studio is een no code-ontwikkelomgeving en Studio Pro een low code-ontwikkelomgeving. Daarmee kunnen ontwikkelaars van verschillende kennisniveaus samenwerken en applicaties bouwen om hun organisaties verder te digitaliseren. De twee oplossingen gebruiken volgens Mendix dezelfde visuele taal, metadata modellen en repository op een enkel uniform platform. [&#8230;]
