Microsoft heeft versie 16.0.1 van Visual Studio 2019 uitgebracht, en enkele dagen eerder ook al versie 15.9.11 van Visual Studio 2017. Deze populaire programmeerontwikkelomgeving beschikt over handige opties om het programmeren in onder andere Visual C++, Visual Basic, C#, F#, Python en R makkelijker te maken. De hele waslijst veranderingen van de 2019-uitgave kun je nalezen in de bijbehorende releasenotes, voor de 2017-uitgave zijn er ook releasenotes.
