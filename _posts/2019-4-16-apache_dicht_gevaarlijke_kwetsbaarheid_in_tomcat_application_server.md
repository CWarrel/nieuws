De Apache Software Foundation heeft een update uitgebracht voor zijn Tomcat application server-software, waarin een belangrijke kwetsbaarheid gedicht wordt. Het gaat om een remote code execution-kwetsbaarheid, schrijft IT Pro. Tomcat wordt ontwikkeld en aangeboden onder open source-licenties. Het is een Servlet-container voor Java-apps, ontworpen om een web server-omgeving te bieden die alleen maar bestaat uit [&#8230;]
